접힌 사이드바에서 카테고리·빠른 실행 아이콘의 hover 영역·크기·radius·간격·정렬을 전부 size-9(36px) 정사각형 + rounded-md + flex items-center + gap-0.5로 통일.

## 배경

- 카테고리: w-full + py-2 (가변 너비 직사각형), rounded-md
- 빠른 실행: size-9 (36px 정사각형), rounded-lg
- hover 네모칸 크기·모양이 달라서 시각적으로 어긋남
- 세로 정렬도 부모 padding 차이로 안 맞음

## 변경

| 항목 | Before | After |
|------|--------|-------|
| 카테고리 접힌 버튼 | w-full py-2 rounded-md | size-9 rounded-md |
| 빠른 실행 접힌 버튼 | size-9 rounded-lg | size-9 rounded-md |
| 부모 정렬 | px-1 (좌측 패딩) | flex-col items-center (중앙) |
| 간격 | space-y-0.5 vs gap-1 | 둘 다 gap-0.5 |
| 아이콘 크기 | 카테고리 16px, 빠른실행 14px | 전부 16px |

## 원칙 추가

`memory/rules/dashboard-design-system.md` §4에 기록해야 할 사항:
- 접힌 사이드바 아이콘은 반드시 size-9 정사각형 + rounded-md
- 아이콘 크기는 16px 통일
- 부모는 flex-col items-center로 중앙 정렬

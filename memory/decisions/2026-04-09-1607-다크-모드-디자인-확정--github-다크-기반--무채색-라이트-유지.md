## 배경

- 1차 시도: 인디고/바이올렛 포인트 → 사용자 피드백 "AI 스럽고 흔해빠진 느낌"
- 2차 시도: 완전 무채색 → 사용자 피드백 "가독성 별로, 깔끔하지도 않음, 밋밋함"
- 3차 시도: GitHub 다크 기반 → 확정

## 확정 다크 모드 수치

| 항목 | OKLCH |
|------|-------|
| background | 0.145 / 0.014 / 256 |
| card | 0.19 / 0.014 / 256 |
| border | 0.30 / 0.01 / 252 |
| foreground | 0.94 / 0.01 / 250 |
| muted-foreground | 0.62 / 0.014 / 252 |
| sidebar | 0.155 / 0.014 / 256 |

## 핵심 원칙

- 블루 틴트 hue 252-256 (인디고/바이올렛 아님)
- chroma 0.01-0.014 수준 (미세하지만 차가운 느낌)
- bg↔card L 격차 0.045, bg↔border L 격차 0.155
- 라이트 모드는 완전 무채색(hue=0) 유지
- AI 스러운 그라데이션/글로우/shadow-color 금지

## 기각된 방향

- 인디고/바이올렛 포인트 (AI 느낌)
- 완전 무채색 다크 (평면적, 가독성 부족)

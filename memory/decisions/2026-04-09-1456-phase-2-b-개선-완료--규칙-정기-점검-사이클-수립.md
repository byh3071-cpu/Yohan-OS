v2 Phase 2-B 차트 데이터 버그 수정(Date 객체 파싱), 카테고리/소스 분포 차트 분리, 세션로그 기반 체인지로그, 가로 시간축 타임라인 구현. 규칙 개정 점검 사이클(매세션/주간/월간) 수립.

## 변경 내역

### 데이터 버그
- gray-matter가 ISO 날짜 문자열을 Date 객체로 파싱 → `typeof === 'string'` 실패
- `extractDate()` 함수: Date 객체/string/파일명 순서로 fallback
- `created`, `published` 필드도 지원

### 차트 개선
- 도메인 분포(태그 기반, 90% 기타) → 카테고리 분포(100% 커버) + 소스 분포(RSS source_name) 2개로 분리
- 각 차트에 맥락 텍스트(핵심 수치 한 줄) 추가

### 타임라인
- 세로 목록 → 가로 시간축 + 날짜별 막대 + 클릭→상세
- 세션 로그 기반 체인지로그 (한국어 요약)

### 규칙 점검
- `memory/rules/rule-review-cycle.md` 생성
- 8개 규칙에 프론트매터 없음 확인 → 다음 세션 일괄 추가 권장

# 개발자의 커서 3 후기, 그들은 왜 따라가기 바쁠까?

저는 커서를 약 2년 동안 사용해 왔습니다. 개인 비용으로 결제해 쓸 만큼, 커서를 좋아하는 편입니다. 특히 커서의 Tab 자동완성 기능을 인상 깊게 사용했는데요. 제가 작성하려던 코드의 흐름을 자연스럽게 이어주기도 하고, 때로는 미처 생각하지 못한 로직이나 수정이 필요한 지점을 먼저 짚어주기도 했습니다. 커서가 해당 라인으로 이동해 Tab 입력을 기다리고 있을 때면, 매번 감탄하며 즐겁게 코딩하곤 했습니다. 이처럼 기대와 믿음을 주던 커서가 이번에 커서 3를 출시했습니다. 개인적으로는 커서가 다른 도구와 무엇으로 경쟁할지, 어떤 차별점을 내세울지 궁금했습니다. 그래서 2주가량 클로드 코드를 잠시 내려놓고, 커서 3만 사용해 사이드 프로젝트를 구현해 보았습니다.

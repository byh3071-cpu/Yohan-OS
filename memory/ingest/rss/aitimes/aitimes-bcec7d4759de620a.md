# X, '그록' 기반 자동 번역·사진 편집 기능 출시

X가 자동 번역과 AI 이미지 편집 기능을 도입하며 사용자 경험 강화에 나섰다. X는 8일(현지시간) 게시물을 자동으로 번역하는 기능과 자연어 명령을 통해 이미지를 수정할 수 있는 새로운 사진 편집 기능을 출시했다. 두 기능 모두 xAI의 ‘그록’ 모델을 기반으로 구현됐다.X는 게시물 자동 번역 기능을 전 세계에 차례로 적용하기 시작했다. 사용자는 설정 아이콘을 통해 특정 언어에 대한 자동 번역 기능을 선택할 수 있다.이는 다양한 언어권 사용자 간 소통을 확대하고 콘텐츠 도달 범위를 넓히려는 조치로, 레딧 등 다른 플랫폼들도 비슷한

# "망치 대신 AI"…전남, 대불산단에 '조선산업 AX 실증센터' 세운다

전라남도가 조선산업의 AI 기반 생산공정 전환에 본격 착수했다.전남은 산업통상자원부의 ‘조선해양 생산공정혁신(AX) 지원 기반구축’ 사업에 선정돼 AI 기반 조선공정 혁신 인프라를 구축하고 생산공정 자동화율을 높일 계획이라고 6일 밝혔다.이번 사업은 기존 기계 중심 자동화를 넘어 AI가 공정을 스스로 최적화하는 지능형 생산 방식 전환을 겨냥했다. 전남은 이를 통해 대불산단을 조선산업 AX 전초기지로 키우겠다는 구상을 드러냈다.이에 따라 전남은 7월부터 2030년까지 국비 100억원을 포함해 총 231억원을 투입해 영암 대불자유무역지

# 챗GPT 기본 모델 'GPT-5.5 인스턴트'로 교체..."지능↑ 환각↓"

오픈AI가 '챗GPT'의 기본 모델을 ‘GPT-5.5 인스턴트(Instant)’로 전면 교체하며 정확성과 개인화 기능을 강화했다. 오픈AI는 5일(현지시간) 기존 ‘GPT-5.3 인스턴트’를 대체하는 ‘GPT-5.5 인스턴트’를 출시했다. 우선 웹 환경의 '챗GPT 플러스' 및 '프로' 사용자에게 제공되며, 모바일과 무료, 기업 사용자로도 차례로 확대될 예정이다.GPT-5.5 인스턴트는 특히 법률, 의료, 금융 등 민감한 분야에서의 환각 문제를 크게 줄인 것이 특징이다.내부 평가에서 이 모델은 고위험 질문에 대해 기존 대비 52.5

# 미스트랄, '바이브' 통해 원격 에이전트·미스트랄 미디엄 3.5 출시

미스트랄 AI가 코딩 에이전트 생태계 확장을 위한 대규모 업그레이드를 단행했다. 미스트랄은 1일(현지시간) 코딩 에이전트 플랫폼 바이브(Vibe)에 ‘원격(클라우드) 에이전트’ 기능을 도입하고, 차세대 모델 ‘미스트랄 미디엄 3.5(Mistral Medium 3.5)’를 공개했다.바이브는 명령줄 인터페이스(CLI)를 통해 작동하는 코딩 에이전트로, 코드 작성과 리팩토링, 테스트 생성, CI 오류 분석 등 다양한 개발 작업을 자동으로 수행한다. 기존에는 모든 작업이 로컬 환경에서 실행됐지만, 이번 업데이트로 클라우드 기반 실행이 가능

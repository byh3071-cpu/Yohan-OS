# “우리는 오늘, 이순신이다”...역대급 흥행 기록한 여수거북선축제

여수시가 개최한 제60회 여수거북선축제가 3일간 31만명의 방문객을 끌어모았다고 6일 밝혔다.지난 1일부터 3일까지 이순신광장과 중앙동 일원에서 열린 이번 축제에서는 지난해 22만명보다 41% 증가한 관람객 수를 기록했다.축제는 “우리는 오늘, 이순신이다!”를 슬로건으로 내걸고 역사 체험과 시민 참여형 프로그램을 결합한 방식으로 운영됐다.올해 행사는 축제 상징성과 현장 참여도를 동시에 끌어올리는 데 초점이 맞춰졌다. 대표 프로그램인 ‘통제영길놀이’에는 개최 연도에 맞춘 1967명이 참여하며 대규모 거리 퍼포먼스를 연출했다.전라좌수영

# [게시판] 메가존클라우드, 지난해 매출 1.75조로 흑자 전환 등 단신

■ 메가존클라우드(대표 염동훈)는 2025년 실적 감사보고서를 공시, 연간 실적 흑자 전환에 돌입했다고 밝혔다. 연결 기준 매출은 전년 대비 27.9% 성장한 1조7496억원을 기록했으며 당기순이익은 82억원, 영업이익도 흑자를 달성했다. AWS 관련 비즈니스와 구글 클라우드, 구글 워크스페이스 등의 매출이 빠르게 성장하며 실적을 견인했다는 설명이다.■ 인라이플(대표 한경훈)은 지난해 연결 기준 매출 784억원, 영업이익 37억원을 달성했다고 밝혔다. 그중 전체 매출의 53.9%는 서비스형 소프트웨어(SaaS) 사업 부문에서 발생했

## 목표
- 10개 용어가 서로 `[[링크]]`로 연결되어, 한 페이지에서 2~4개 관련 용어로 점프할 수 있는 **최소 지식 그래프**를 만든다.

## 범위
- **하는 것:** 각 md 파일에 `## 관련 용어` 섹션 추가, Starlight 링크 문법으로 상호 참조.
- **안 하는 것:** 백링크 자동 집계, Graph View UI.

## 연결 관리 규칙 (vision.md 제약 반영)
- 파일당 **직접 링크 2~4개만** — 초과 시 혼잡.
- 관계 종류 명시:
  - **상위 개념**: 이건 뭐의 일부인가 (예: 하네스 → 컨텍스트엔지니어링)
  - **재료**: 이걸 하려면 뭐가 필요한가 (예: RAG → LLM + 토큰)
  - **결과**: 이걸 하면 뭐가 되나 (예: 하네스 → 바이브코딩)
- 나머지는 Starlight 기본 검색이 처리.

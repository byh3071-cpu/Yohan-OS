# 세션 로그 디렉토리

에이전트(Cursor AI)가 **세션 종료 시** 자동으로 남기는 구조화 로그.

## 파일명 규칙

`session-YYYY-MM-DD-HHMM.md`

## 용도

- 대시보드 "에이전트 라이브 피드" 데이터 소스
- 오늘의 브리핑에서 "어제 에이전트가 한 일" 요약 재료
- 세션 간 맥락 전달 (이전 세션에서 뭘 했는지 빠르게 파악)

## 작성 규칙

`memory/rules/session-log.md` 참조.
